# Getting Started

## Quick start

![Getting started with Compass](/images/welcome-1.png)

In your Compass channel mention `@compass` and ask a question like:

- *"What data can you query?"*
- *"What are our top opportunities this quarter?"*
- *"I'm a sales leader, what data can you show me?"*
- *"I'm a product manager, how can this tool help me?"*

## Collaborate in Compass

![Team collaboration in Compass](/images/welcome-3.png)

You can use Compass alongside your teammates in the same thread.

#### How it works:

1. Someone asks a question in Slack.
2. Another person can reply in the same thread with follow-ups.
