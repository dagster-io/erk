# Slack App Permissions

## What will Compass be able to do?

**`app_mentions:read`**
View messages that directly mention `@Compass` in conversations that the app is in

**`channels:join`**
Join public channels in a workspace

**`chat:write`**
Send messages as `@Compass`

**`files:write`**
Upload, edit, and delete files as Compass

**`chat:write.customize`**
Send messages as `@Compass` with a customized username and avatar

**`commands`**
Add shortcuts and/or slash commands that people can use

---

## What will Compass be able to view?

**`channels:history`**
View messages and other content in public channels, private channels, direct messages, and group direct messages that Compass has been added to

**`groups:history`**
View messages and other content in public channels, private channels, direct messages, and group direct messages that Compass has been added to

**`im:history`**
View messages and other content in public channels, private channels, direct messages, and group direct messages that Compass has been added to

**`mpim:history`**
View messages and other content in public channels, private channels, direct messages, and group direct messages that Compass has been added to

**`channels:read`**
View basic information about public channels in a workspace

**`files:read`**
View files shared in channels and conversations that Compass has been added to

**`groups:read`**
View basic information about private channels that Compass has been added to

**`mpim:read`**
View basic information about group direct messages that Compass has been added to

**`users:read`**
View people in a workspace
