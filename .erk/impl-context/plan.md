# Fix hardcoded workspace packages in activation script

## Context

`render_activation_script()` in `src/erk/cli/activation.py:207` hardcodes:
```bash
uv pip install --no-deps --quiet -e . -e packages/erk-shared -e packages/erk-statusline
```

This only works in the erk monorepo. Consumer repos (e.g., dagster-compass) don't have `packages/erk-shared` or `packages/erk-statusline` — they install those from PyPI. The activation script produces a noisy error on every worktree activation:
```
error: Distribution not found at: file:///...packages/erk-shared
```

## Approach

Replace the hardcoded package list with dynamic workspace member discovery **at shell execution time**. Use an inline Python snippet to parse `pyproject.toml` `[tool.uv.workspace].members` and only install packages that exist on disk.

**Why execution-time over render-time:** The activation script is written to `.erk/bin/activate.sh` once during worktree creation but sourced many times. Discovering at execution time means:
- Always reflects current worktree state (packages added/removed)
- No need to thread a new parameter through the 9+ call sites of `render_activation_script`
- Works even if the script was generated by an older erk version

**Why Python over pure shell:** `pyproject.toml` uses TOML with glob patterns for workspace members (e.g., `packages/*`). Python has `tomllib` (stdlib since 3.11) and `glob` — both available since erk requires Python 3.11+. Shell-only parsing of TOML + glob expansion would be fragile.

## Changes

### 1. `src/erk/cli/activation.py` — Replace hardcoded line (~207)

Replace:
```bash
uv pip install --no-deps --quiet -e . -e packages/erk-shared -e packages/erk-statusline
```

With a shell block that:
1. Uses inline Python to read `pyproject.toml` and discover workspace member directories
2. Falls back to just `-e .` if parsing fails or no workspace is configured
3. Runs a single `uv pip install --no-deps --quiet` with the discovered packages

```bash
# Discover workspace members from pyproject.toml
__erk_editable=$(python3 -c "
import tomllib, pathlib, glob
parts = ['-e .']
p = pathlib.Path('pyproject.toml')
if p.exists():
    cfg = tomllib.loads(p.read_text())
    for pat in cfg.get('tool', {}).get('uv', {}).get('workspace', {}).get('members', []):
        for m in sorted(glob.glob(pat)):
            if (pathlib.Path(m) / 'pyproject.toml').exists():
                parts.append(f'-e {m}')
print(' '.join(parts))
" 2>/dev/null || echo "-e .")
uv pip install --no-deps --quiet $__erk_editable
```
